Σκοπός:
Ο σκοπός κατά την εγγραφή αυτού του update, είναι η ανάπτυξη μίας εφαρμογής που θα παρέχει στο χρήστη υπηρεσίες πλοήγησης σε μέρη πολιτιστικού ενδιαφέροντος.

Τεχνολογίες:
Οι τεχνολογίες που μπορούν να χρησιμοποιηθούν για την επίτευξη του σκοπού, κατά τη στιγμή εγγραφής αυτού του update μπορούν να χωριστούν σε client και server side.
Client: React Native για την ανάπτυξη του front-end της εφαρμογής, Geolocation για την εύρεση της θέσης του χρήστη, Google Maps API για την απεικόνιση σε χάρτη της τοποθεσίας του χρήστη και των σημείων ενδιαφέροντος
Server: Βάση Δεδομένων για στοιχεία σύνδεσης χρήστη κλπ, Google Places API για εύρεση σημείων ενδιαφέροντος κοντά στην τοποθεσία του χρήστη.

Επιπρόσθετα χαρακτηριστικά:
Μερικά χαρακτηριστικά που έχω σκεφτεί για να αυξηθεί η διαδραστικότητα με τον χρήστη, είναι η δημιουργία ενός score που θα αυξάνεται όποτε ο χρήστης επισκέπτεται ένα σημείο ενδιαφέροντος. Ανάλογα με την επισκεψιμότητα του κάθε σημείου (δεδομένο που νομίζω μπορώ να αντλήσω από το Google Places), ο χρήστης θα επιβραβεύεται με ανάλογο score. Ενδεχομένως να μπορεί να δημιουργηθεί και ένα leaderboard για κάθε πόλη που θα περιέχει τους top χρήστες. Έτσι, πέρα από την αλληλεπίδραση με την πόλη και την γνώση της ιστορίας και του πολιτισμού της που κερδίζει ο χρήστης, το παιχνίδι αποκτά και έναν ανταγωνιστικό χαρακτήρα.
